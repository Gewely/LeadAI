# LeadFlow AI

LeadFlow AI is a lead-to-sale tracking SaaS application concept for campaign management, lead handling, sales outcomes, and performance analytics.

## Repository Contents
- `index.html`, `styles.css`, `app.js`: runnable front-end prototype implementing the requested flows.
- `BUBBLE_IMPLEMENTATION.md`: full Bubble build blueprint (data types, roles, privacy, workflows, pages, and QA checklist).

## Quick Start
Open `index.html` in your browser.
